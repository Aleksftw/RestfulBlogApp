# RESTFulBlogApp
---
Full stack app built following The Web Developer Bootcamp by Colt Steele (on Udemy).

The following tech stack was utilised:
- [x] NodeJS/express  
- [x] MongoDB/mongoose  
- [x] Semantic UI  

Functionalities:  
- [x] User can make new blog posts  
- [x] User can modify and delete existing entries
